#### 371. Sum of Two Integers

<span style="color:#deb800">***Medium***</span>
___

Given two integers `a` and `b`, return _the sum of the two integers without using the operators_ `+` _and_ `-`.
___

**Example 1:**

>**Input:** a = 1, b = 2
>
>**Output:** 3

**Example 2:**

>**Input:** a = 2, b = 3
>
>**Output:** 5
___

**Constraints:**

*   `-1000 <= a, b <= 1000`
